# Классификация транзакций с дисбалансом классов

## Что делает скрипт
- Объединяет Category и Subcategory в единый класс.
- Строит признаки по тексту Note, способу оплаты Mode, сумме Amount и времени.
- Обучает три модели:
  1) class_weight="balanced"
  2) SMOTE
  3) Иерархическое взвешивание по Category
- Считает macro F1 по всем классам и F1 для 10 редких.

## Как запустить
pip install -r requirements.txt
python transactions_imbalanced.py

## Выход
В консоль выводятся:
- общее число классов;
- список редких классов;
- Macro F1 для трёх методов;
- F1 для 10 редких классов для каждого метода.
